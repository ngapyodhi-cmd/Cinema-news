      네이버 뉴스 연동 (선택)

      API 키를 설정하면 Google과 함께 네이버 뉴스도 불러옵니다.

      1. 네이버 개발자 센터에서 앱 등록
      2. 검색 API 사용 · Client ID·Secret 발급
      3. Render → Environment에 아래 변수 추가 후 재배포
      NAVER_CLIENT_ID=발급_ID
NAVER_CLIENT_SECRET=발급_Secret

      불러오기 실패

      터미널에서 node server.js로 서버를 실행했는지 확인하세요.
